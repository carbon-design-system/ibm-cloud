# getDateLocaleFormat

## Description

Takes a locale and returns its date format. Similar to `Intl.DateTimeFormat().formatToParts` however
it is supported on all browsers. Falls back to `en`.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { getDateLocaleFormat } from '@console/pal/Utilities';
    ```

3.  Use the utility

    ```js
    getDateLocaleFormat(input);
    ```
