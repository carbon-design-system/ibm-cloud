# getLocale

## Description

The getLocale utility will take in any locale and return a support IBM Cloud locale.
For example, if you pass in 'en-us', the getLocale utility will return 'en'. This is
a good resource for passing in the user's set locale, `window.navigator.language`, and
return IBM Cloud's supported locale for that language.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { getLocale } from '@console/pal/Utilities';
    ```

3.  Use the utility

    ```js
    getLocale(input);
    ```
