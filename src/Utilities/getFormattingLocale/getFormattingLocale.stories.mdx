# getFormattingLocale

## Description

Get a locale to use for formatting dates and numbers. Note that this is different from the language used for translation. This locale is based on the current browser locale of the user, regardless of supported translation language.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { getFormattingLocale } from '@console/pal/Utilities';
    ```

3.  Use the utility

    ```js
    getFormattingLocale(input);
    ```
