# formatDateNumeric

## Description

Takes in a Date object and returns a numeric string based on the user's locale.

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { formatDateNumeric } from '@console/pal/Utilities';
    ```

3.  Use the utility

    ```js
    formatDateNumeric(input);
    ```
