# getDateLocalePlaceholder

## Description

Takes a locale and returns its date placeholder. Works based off `getDateLocaleFormat` and converts to dd mm yyyy

## Usage

1.  Install the [@console/pal](https://github.ibm.com/ibmcloud/pal#how-to-install-and-use) library
2.  Import the utility

    ```js
    import { getDateLocalePlaceholder } from '@console/pal/Utilities';
    ```

3.  Use the utility

    ```js
    getDateLocalePlaceholder(input);
    ```
